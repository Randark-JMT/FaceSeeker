# FaceAtlas - 人脸识别与聚类分析系统

功能强大的人脸识别系统，并可根据识别到的人脸数据进行归类计算，将同一个体的出镜图像进行关联计算并展示

## 核心特性

- 基于 OpenCV 实现的图像识别模型
- 基于多指标（Laplacian 方差，Tenengrad 梯度、FFT）所实现的多指标融合的清晰度评估算法
- 支持单张图像中多人脸的检测与识别
- 基于人脸结构特征向量所构建的聚类分析算法
- 基于系统硬件 CPU 核心数，自动调整最大工作线程，确保在大规格数据集中保持较高的计算资源利用率
- 可兼容多种计算后端，如 CUDA（需自编译计算依赖库），OpenCL等，优化计算效率
- 支持多项目管理，通过不同 Sqlite 数据库，对不同项目的识别数据进行持久化管理
